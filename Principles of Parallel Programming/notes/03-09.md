# 内存层级

pageable, pin(连续的内存地址, 分配之后os不会使用它, dma加上映射表), device memory(每个gpu不同),
L2cache(fermi 数据隔一层), constant memory(速度很快, 调用过程中不能更改),
shared memory(L1cache, 多核共享), register(放的东西少)
